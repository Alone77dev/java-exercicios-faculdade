# ☕ Java Exercícios - Faculdade

Repositório com exercícios práticos de Java desenvolvidos ao longo do curso de graduação. O projeto está organizado em dois grandes módulos: **Java Imperativo** e **POO (Programação Orientada a Objetos)**.

---

## 📁 Estrutura do Projeto

```
src/
├── javaimperativo/
│   ├── entradasaida/         # Exercícios de entrada e saída de dados
│   ├── estruturadecisao/     # Estruturas condicionais (if, switch)
│   ├── estruturarepeticao/   # Laços de repetição (for, while, do-while)
│   └── metodosestaticos/     # Métodos estáticos e funções auxiliares
│
└── poo/
    ├── array/                # Manipulação de arrays e vetores
    ├── fundamentos/          # Conceitos base de OOP (classes, objetos, atributos)
    └── modelos/              # Modelagem com herança, encapsulamento e polimorfismo
```

---

## 📚 Conteúdos Abordados

### 🔷 Java Imperativo
Exercícios focados na lógica de programação e nos fundamentos da linguagem Java:

- **Entrada e Saída** — Leitura via `Scanner`, formatação e exibição de dados com `System.out`
- **Estrutura de Decisão** — Uso de `if/else`, `else if` e `switch/case`
- **Estrutura de Repetição** — Laços `for`, `while` e `do-while`, incluindo laços aninhados
- **Métodos Estáticos** — Criação e chamada de métodos, passagem de parâmetros e retorno de valores

### 🔶 POO — Programação Orientada a Objetos
Exercícios voltados aos pilares da orientação a objetos:

- **Arrays** — Declaração, inicialização, percurso e manipulação de arrays unidimensionais e bidimensionais
- **Fundamentos** — Encapsulamento, herança, polimorfismo e construtores
- **Modelos** — Criação e modelagem de classes, objetos e atributos

## ▶️ Como Executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/java-exercicios-faculdade.git
   ```

2. **Abra no IntelliJ IDEA** (ou outra IDE de sua preferência).

3. **Navegue até o exercício desejado** dentro de `src/` e execute o arquivo `.java` correspondente.

> Não há dependências externas. Basta ter o JDK instalado.
